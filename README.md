# Velocidad de acceso a memoria

Localidad es un concepto importante cuando se desea un desempeño óptimo y consiste en que mientras mas contigua se encuentre la información en la memoria mientras se accede a ella mejor ya que existe más posibilidad de que se encuentre en memoria de más rápido acceso.
